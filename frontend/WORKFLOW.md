# 工作流程说明

## 🎯 完整工作流程

### 1️⃣ 创建产品
```
侧边栏 → 📦 产品管理 → 新建产品
输入产品名称：精华液
点击 [创建]
```

### 2️⃣ 搜索关键词
```
点击产品卡片"精华液" → 进入产品详情页

新增搜索：
- 关键词：精华液推荐
- 数量：20
点击 [开始抓取]

→ 自动跳转到"当前工作"页面
```

### 3️⃣ 继续添加更多关键词
```
在"当前工作"页点击 [新增搜索]
→ 返回产品详情页

新增搜索：
- 关键词：平价精华液
- 数量：15
点击 [开始抓取]

再次添加：
- 关键词：美白精华液
- 数量：25
点击 [开始抓取]
```

### 4️⃣ 合并多个关键词到工作区 ⭐新功能

#### 方式一：多选合并加载
```
在产品详情页的"搜索历史"区域：

☑ 精华液推荐 (20条)
☑ 平价精华液 (15条)
☑ 美白精华液 (25条)

点击 [加载到工作区 (3)] 按钮

→ 合并加载 60 条笔记到工作区
→ 顶部显示：精华液 → 精华液推荐 + 平价精华液 + 美白精华液
```

#### 方式二：单独加载
```
点击某条搜索记录的 [单独加载] 按钮
→ 只加载该关键词的笔记
```

### 5️⃣ 在工作区进行创作
```
当前工作页面：

顶部信息栏：
📁 精华液 → 精华液推荐 + 平价精华液 + 美白精华液
笔记数: 60  已选: 0  仿写: 0  正文: 0

左栏：笔记列表
- 每条笔记显示所属关键词标签
- [精华液推荐] 这款精华真的绝了...
- [平价精华液] 学生党必入...
- [美白精华液] 美白效果太好了...

中栏：情绪词 & 场景词分析
- 自动分析所有笔记的关键词

右栏：AI 创作
- 选择笔记 → 点击 [仿写] → 生成标题
- 选择笔记 → 点击 [正文] → 生成内容
```

## 📊 多关键词合并的优势

### 1. 跨关键词对比分析
```
同时查看：
- 精华液推荐：强调"必买"、"绝了"
- 平价精华液：强调"性价比"、"学生党"
- 美白精华液：强调"美白"、"提亮"

→ 发现不同关键词的内容差异
→ 提取共性和特性
```

### 2. 批量内容创作
```
一次性加载多个关键词的笔记
→ 可以选择不同风格的笔记进行仿写
→ 创作更多样化的内容
```

### 3. 全面的情绪词分析
```
合并分析多个关键词的笔记
→ 得到更全面的情绪词和场景词
→ 词库更丰富
```

## 💡 使用技巧

### 技巧 1：批量选择关键词
```
1. 点击 [全选] 快速选中所有搜索记录
2. 或手动勾选想要的关键词
3. 点击 [加载到工作区 (N)] 合并加载
```

### 技巧 2：按关键词筛选笔记
```
在笔记列表中，每条笔记都有关键词标签
→ 可以快速识别笔记来源
→ 便于分类整理和创作
```

### 技巧 3：返回继续添加关键词
```
在当前工作页点击 [新增搜索]
→ 返回产品详情页
→ 添加新关键词
→ 再次合并加载
```

### 技巧 4：单独加载某个关键词
```
如果只想专注于某个关键词
→ 点击该记录的 [单独加载] 按钮
→ 只加载该关键词的笔记到工作区
```

## 🔄 快速操作流程

### 场景：同时分析多个关键词

```
1. 创建产品"精华液"
2. 连续搜索 3 个关键词：
   - 精华液推荐 (20条)
   - 平价精华液 (15条)
   - 美白精华液 (25条)
3. 在产品详情页勾选这 3 个关键词
4. 点击 [加载到工作区 (3)]
5. 在工作区查看 60 条笔记
6. 开始 AI 创作
```

### 场景：切换不同关键词组合

```
1. 当前加载：精华液推荐 + 平价精华液
2. 想换成：精华液推荐 + 美白精华液

操作：
- 点击 [返回产品]
- 重新勾选想要的关键词组合
- 点击 [加载到工作区]
```

## ✅ 核心功能总结

| 功能 | 位置 | 说明 |
|------|------|------|
| 新建产品 | 产品管理页 | 创建产品分类 |
| 搜索关键词 | 产品详情页 | 抓取笔记数据 |
| 多选合并 | 产品详情页 | 勾选多个关键词合并加载 |
| 单独加载 | 产品详情页 | 加载单个关键词的笔记 |
| 查看标签 | 笔记列表 | 识别笔记所属关键词 |
| 新增搜索 | 当前工作页 | 快速返回添加新关键词 |
| 返回产品 | 当前工作页 | 返回产品详情页重新选择 |
| AI 创作 | 当前工作页 | 标题仿写 + 正文创作 |

## 🎯 最佳实践

1. **先规划产品结构**
   - 按产品类别创建（如：精华液、面霜、口红）
   - 每个产品下搜索多个相关关键词

2. **批量搜索关键词**
   - 一次性搜索多个相关关键词
   - 积累足够的笔记数据

3. **灵活组合分析**
   - 根据需要选择不同关键词组合
   - 进行跨关键词对比分析

4. **持续迭代优化**
   - 根据分析结果添加新关键词
   - 不断丰富词库和内容
